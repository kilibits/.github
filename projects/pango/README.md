## What is Ember?

We have reimagined the concept of "kikoba" as a standalone application, where you can pay in with mobile money, debit cards, request and manage loans.

We strongly believe this should never be tied to your network provider, the money flow should be open and visible to all members

### ðŸ“± Core Features
- **Member Management** - Track group members, roles, and contact information
- **Contribution Tracking** - Record and monitor monthly contributions
- **Loan Management** - Request, approve, and track loans with automatic interest calculation
- **Mobile Money Integration** - Pay via M-Pesa, Airtel Money, Tigo Pesa, and Halopesa
- **Reports & Analytics** - Visual charts showing group financial health
- **Notifications** - Real-time alerts for payments, approvals, and reminders
